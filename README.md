## Tasks Completed

### Level 3: Pizza Delivery Application

Tech Stack: MERN (MongoDB, Express, React, Node.js)

Folder: Level3_Pizza_Delivery_App

A full-stack pizza ordering platform with custom pizza building, payment integration, real-time order tracking, and admin management.

### Key Features

- User and admin authentication with email verification and password reset
- Custom pizza builder (5 bases, 5 sauces, cheese, veggies, meats)
- Payment integration (Razorpay-style test mode)
- Order management with automatic stock deduction
- Real-time order status updates (Socket.io)
- Low-stock email alerts to admin
- Admin dashboard with inventory and order management
- Live pizza visualization with toppings

See the project folder for full details, setup instructions, and demo links.
